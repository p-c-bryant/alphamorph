CLEAN → DIRTY

Objective

Transform the Start Word into the End Word by creating a sequence of valid intermediate words. Try to reach the End Word in as few moves as possible!

Gameplay Rules

Change One Letter: On each turn, change exactly one letter in the current word to form a new valid word. Example: "COLD → HOLD" is valid, because only one letter has been changed between the two words (changing C to H).

You May Reorder Letters: You may reorder the letters of the current word to form a new word, as long as you only change one letter. Example: HALE → HEAT is valid, because there is only one letter that is different between the two words (changing "L" to "T"), even though the letters are reordered.

No Multiple Letter Changes: You can only change one letter per turn. If the word contains duplicates of a letter, you can still only change one instance of that letter in a single turn. Example: COOL → CALL is invalid, because both "O"s are changed at the same time. A valid alternative would be COOL → COAL, because only one "O" is changed.

Example Gameplay:

Start Word: COLD

End Word: HEAT

Valid Sequence:

  1. COLD → HOLD (Change C to H)
  2. HOLD → HOLE (Change D to E)
  3. HOLE → HALE (Change O to A)
  4. HALE → HEAT (Change L to T and reorder)

COLD → HOLD → HOLE → HALE → HEAT
